# Ambashaw App

**Ambashaw App** is a simple and user-friendly business finance application designed to help small businesses track income, expenses, and remaining balance in Ethiopian Birr. It allows users to record transactions easily and monitor cash flow for better financial management.

---

## Features

- Track daily **income** and **expenses**
- View **recent transactions** at a glance
- Monitor **remaining balance**
- Simple, clean, and intuitive interface
- Responsive design for mobile and desktop
